You run VEP on your VCF. It finishes. You open the output and see something like this in the INFO column:

```

```

Congratulations. That is one variant. One.

VCF's are tab-delimited and their INFO column is like a dictionary delimited by semi-colons with equal-signs to signal the keys and values. The key "CSQ" is VEP's Consequence (CSQ) field. It is pipe-delimited within each transcript annotation, and comma-delimited between transcripts. A single variant can have dozens of transcripts (like the one above), but a multi-allelic site (a region with multiple ALT alleles) can have dozens of variants. By the time you have a whole cohort, you are sitting on a wall of text that looks like it was generated by someone who hates you.

This post walks through what the CSQ field actually contains and how to parse it into something usable.

## What VEP Is Doing

VEP (Variant Effect Predictor) takes your variant calls and annotates each one against every overlapping transcript in the reference genome. For each transcript, it produces a consequence — things like `missense_variant`, `synonymous_variant`, `stop_gained` — along with a pile of metadata about the gene, the transcript, the protein change, and prediction scores from tools like SIFT and PolyPhen.

All of that gets stuffed into a single INFO field called CSQ.

The VCF header tells you the field order:

```
##INFO=<ID=CSQ,Number=.,Type=String,Description="Consequence annotations from Ensembl VEP. Format: Allele|Consequence|IMPACT|SYMBOL|Gene|Feature_type|Feature|BIOTYPE|EXON|INTRON|HGVSc|HGVSp|cDNA_position|CDS_position|Protein_position|Amino_acids|Codons|Existing_variation|DISTANCE|STRAND|FLAGS|SYMBOL_SOURCE|HGNC_ID|CANONICAL|SIFT|PolyPhen">
```

So the format is: `Allele|Consequence|IMPACT|SYMBOL|...` repeated for every transcript, joined by commas.

## Splitting the String Into a DataFrame

The first step is to split that wall of text into a structured table. Given a row from your variant dataframe where `row["csq"]` holds the raw CSQ string:

```python
csq_cols = [
    "Allele","Consequence","IMPACT","SYMBOL","Gene","Feature_type","Feature",
    "BIOTYPE","EXON","INTRON","HGVSc","HGVSp","cDNA_position","CDS_position",
    "Protein_position","Amino_acids","Codons","Existing_variation","DISTANCE",
    "STRAND","FLAGS","SYMBOL_SOURCE","HGNC_ID","CANONICAL","SIFT","PolyPhen"
]

csq_rows = [r.split("|") for r in csq_str.split(",")]
all_csq_df = pd.DataFrame(csq_rows, columns=csq_cols)
```

For the example CSQ string above, this produces a dataframe with two rows — one per transcript — each with 26 columns. That is already more manageable.

From here we filter down to the gene we care about (in this case, APOE):

```python
all_csq_df = all_csq_df.loc[all_csq_df["SYMBOL"] == "APOE", :].copy()
```

And compute the most severe consequence for each transcript. The `Consequence` field can itself be `&`-delimited when a variant hits multiple consequences in one transcript (e.g., `missense_variant&splice_region_variant`), so we need a severity ranker:

```python
all_csq_df['most_severe_consequence'] = all_csq_df['Consequence'].apply(
    lambda x: get_most_severe_consequence(x)
)
```

## The Allele Matching Problem

Here is where it gets complicated. VEP normalizes alleles differently than your VCF does.

In a VCF, a deletion might be represented as:

```
REF: ACGT
ALT: A
```

But in the CSQ field, VEP may represent the same deletion's allele as `-` (a dash), because it strips the anchor base. An insertion works the opposite way — VEP may store just the inserted sequence without the anchor.

So you cannot simply join on the ALT column directly. You have to detect the mismatch and patch the allele strings before merging.

For insertions (ALT is longer than REF), VEP's allele is missing the anchor base — prepend REF to recover the full ALT:

```python
if len(missed_alt) > len(missing_ref):
    csq_copy['Allele'] = csq_copy['Allele'].apply(lambda x: missing_ref + x)
```

For deletions (ALT is shorter than REF), VEP represents the deleted sequence as `-` — replace the dash with the actual ALT:

```python
elif len(missed_alt) < len(missing_ref):
    csq_copy['Allele'] = csq_copy['Allele'].str.replace("-", missed_alt)
```

## Multi-Allelic Sites

A multi-allelic site is a position where more than one alternate allele was observed. In a VCF, this is a single row with a comma-separated ALT column:

```
REF: A
ALT: T,G
```

VEP annotates each allele separately within the CSQ string, but they all live in the same comma-separated blob. You have to explode the variant row into one row per allele, then merge each allele against its matching CSQ entries:

```python
if "," in rowX["alt"]:
    alt_l = rowX["alt"].split(",")
    varid_l = rowX["var_id"].split(";")

    row_copy = rowX.copy()
    row_copy["alt"] = alt_l
    row_copy["var_id"] = varid_l
    split_rows_df = pd.DataFrame([row_copy]).explode(['alt', 'var_id']).reset_index(drop=True)

    merged_df = pd.merge(
        split_rows_df,
        all_csq_df,
        left_on="alt",
        right_on='Allele',
        how='inner'
    )
```

After the merge, any allele that didn't match directly gets the allele normalization treatment described above.

## Picking One Consequence Per Allele

After merging, each allele may still have multiple rows — one per overlapping transcript. You need to pick one. The priority order is:

1. **Most severe consequence** — rank by a consequence severity list (e.g., `stop_gained` > `missense_variant` > `synonymous_variant`)
2. **Canonical transcript** — prefer `CANONICAL == "YES"` when multiple transcripts share the same top consequence
3. **Protein-coding biotype** — prefer `BIOTYPE == "protein_coding"` if ties remain after canonical filtering

```python
top_conseq = get_most_severe_consequence(
    "&".join(row_alt_df['most_severe_consequence'].unique().tolist())
)
severest_csq_df = row_alt_df.loc[
    row_alt_df['most_severe_consequence'] == top_conseq, :
].drop_duplicates().copy()

if len(severest_csq_df) > 1:
    canonical_mask = severest_csq_df['CANONICAL'] == "YES"
    if canonical_mask.any():
        severest_csq_df = severest_csq_df.loc[canonical_mask, :]

    if len(severest_csq_df) > 1:
        protein_coding_mask = severest_csq_df['BIOTYPE'] == "protein_coding"
        if protein_coding_mask.any():
            severest_csq_df = severest_csq_df.loc[protein_coding_mask, :]
```

If you still have more than one row after all three filters, you have a genuine tie and need to decide what to do — the code above warns and takes the first row.

## The Full Function

Putting it all together, `add_csq_df` takes a single variant row, parses its CSQ string, handles allele normalization, handles multi-allelic sites, and returns a flat dataframe with one row per allele and the winning consequence annotation merged in:

```python
def add_csq_df(rowX, all_var=False):

    csq_str = rowX["csq"]
    csq_cols = [
        "Allele","Consequence","IMPACT","SYMBOL","Gene","Feature_type","Feature",
        "BIOTYPE","EXON","INTRON","HGVSc","HGVSp","cDNA_position","CDS_position",
        "Protein_position","Amino_acids","Codons","Existing_variation","DISTANCE",
        "STRAND","FLAGS","SYMBOL_SOURCE","HGNC_ID","CANONICAL","SIFT","PolyPhen"
    ]
    csq_rows = [r.split("|") for r in csq_str.split(",")]
    all_csq_df = pd.DataFrame(csq_rows, columns=csq_cols)
    all_csq_df = all_csq_df.loc[all_csq_df["SYMBOL"] == "APOE", :].copy()
    all_csq_df['most_severe_consequence'] = all_csq_df['Consequence'].apply(
        lambda x: get_most_severe_consequence(x)
    )

    if all_var:
        return all_csq_df

    if "," in rowX["alt"]:
        alt_l = rowX["alt"].split(",")
        nalts = len(alt_l)
        varid_l = rowX["var_id"].split(";")
        nvarid = len(varid_l)

        if nalts == nvarid:
            row_copy = rowX.copy()
            row_copy["alt"] = alt_l
            row_copy["var_id"] = varid_l
            split_rows_df = pd.DataFrame([row_copy]).explode(['alt', 'var_id']).reset_index(drop=True)

            merged_df = pd.merge(
                split_rows_df,
                all_csq_df,
                left_on="alt",
                right_on='Allele',
                how='inner'
            )
            merged_df_l = []
            if len(merged_df['alt'].unique()) < nalts:
                merged_df_l.append(merged_df)
                missing_alts = [x for x in alt_l if x not in merged_df['alt'].unique()]
                for missed_alt in missing_alts:
                    missing_alt_idx = alt_l.index(missed_alt)
                    missing_varid = varid_l[missing_alt_idx]
                    missing_ref = missing_varid.split(":")[2]
                    csq_copy = all_csq_df.copy()
                    if len(missed_alt) > len(missing_ref):
                        csq_copy['Allele'] = csq_copy['Allele'].apply(lambda x: missing_ref + x)
                        merged_df_miss = pd.merge(
                            split_rows_df.loc[split_rows_df['alt'] == missed_alt, :].copy(),
                            csq_copy,
                            left_on="alt",
                            right_on='Allele',
                            how='inner')
                        merged_df_l.append(merged_df_miss)
                    elif len(missed_alt) < len(missing_ref):
                        csq_copy['Allele'] = csq_copy['Allele'].str.replace("-", missed_alt)
                        merged_df_miss = pd.merge(
                            split_rows_df.loc[split_rows_df['alt'] == missed_alt, :].copy(),
                            csq_copy,
                            left_on="alt",
                            right_on='Allele',
                            how='inner')
                        merged_df_l.append(merged_df_miss)
                    else:
                        print(f"WARNING: MISSING ALT {missed_alt} from REF {missing_ref}")
                merged_df = pd.concat(merged_df_l, ignore_index=True)

            if len(merged_df) == 0:
                print("ERROR: No ALTS")
                return pd.DataFrame()

            final_rows = []
            for alt_j in alt_l:
                row_alt_df = merged_df.loc[merged_df["alt"] == alt_j, :].copy()

                if len(row_alt_df) > 1:
                    top_conseq = get_most_severe_consequence(
                        "&".join(row_alt_df['most_severe_consequence'].unique().tolist())
                    )
                    severest_csq_df = (
                        row_alt_df
                        .loc[row_alt_df['most_severe_consequence'] == top_conseq, :]
                        .drop_duplicates()
                        .copy()
                    )
                    if len(severest_csq_df) > 1:
                        canonical_mask = severest_csq_df['CANONICAL'] == "YES"
                        if canonical_mask.any():
                            severest_csq_df = severest_csq_df.loc[canonical_mask, :]
                        if len(severest_csq_df) > 1:
                            protein_coding_mask = severest_csq_df['BIOTYPE'] == "protein_coding"
                            if protein_coding_mask.any():
                                severest_csq_df = severest_csq_df.loc[protein_coding_mask, :]
                    final_rows.append(severest_csq_df.iloc[0])
                elif len(row_alt_df) == 1:
                    final_rows.append(row_alt_df.iloc[0])
                else:
                    print(f"WARNING: No consequence found for alt {alt_j}")
                    continue

            if len(final_rows) == 0:
                return pd.DataFrame()
            return pd.DataFrame(final_rows)

        else:
            print(f"Error: {alt_l} and {varid_l} do not match lengths.")
            return pd.DataFrame([rowX])

    else:
        if rowX["alt"] not in all_csq_df['Allele'].unique().tolist():
            if len(rowX["alt"]) > len(rowX["ref"]):
                all_csq_df['Allele'] = all_csq_df['Allele'].apply(lambda x: rowX["ref"] + x)
            elif len(rowX["alt"]) < len(rowX["ref"]):
                all_csq_df['Allele'] = all_csq_df['Allele'].str.replace("-", rowX["alt"])

        top_conseq = get_most_severe_consequence(
            "&".join(all_csq_df['most_severe_consequence'].unique().tolist())
        )
        severest_csq_df = (
            all_csq_df
            .loc[all_csq_df['most_severe_consequence'] == top_conseq, :]
            .drop_duplicates()
            .copy()
        )

        if len(severest_csq_df) > 1:
            canonical_mask = severest_csq_df['CANONICAL'] == "YES"
            if canonical_mask.any():
                severest_csq_df = severest_csq_df.loc[canonical_mask, :]
            if len(severest_csq_df) > 1:
                protein_coding_mask = severest_csq_df['BIOTYPE'] == "protein_coding"
                if protein_coding_mask.any():
                    severest_csq_df = severest_csq_df.loc[protein_coding_mask, :]

        row_df = pd.DataFrame([rowX])
        result_df = pd.merge(
            row_df,
            pd.DataFrame([severest_csq_df.iloc[0]]),
            left_on="alt",
            right_on='Allele',
            how='inner')

        if len(result_df) == 0:
            if len(row_df['alt'].values[0]) > len(row_df['ref'].values[0]):
                severest_csq_df_copy = severest_csq_df.copy()
                severest_csq_df_copy['Allele'] = severest_csq_df_copy['Allele'].apply(
                    lambda x: row_df['ref'].values[0] + x
                )
                result_df = pd.merge(
                    row_df,
                    pd.DataFrame([severest_csq_df_copy.iloc[0]]),
                    left_on="alt",
                    right_on='Allele',
                    how='inner')
                if len(result_df) == 0:
                    return pd.DataFrame()
            else:
                return pd.DataFrame()

        return result_df
```

## Why Is This So Much Code?

Because the CSQ field was designed to be comprehensive, not convenient. VEP is doing the right thing — giving you every possible annotation for every possible transcript. The complexity of parsing it is a reflection of how much information it's encoding, not a flaw in VEP itself.

The real gotchas are:

- **Allele normalization**: VEP and your VCF may represent the same allele differently. You have to detect and patch this.
- **Multiple transcripts per allele**: One variant can hit many transcripts. You need a principled way to pick one.
- **Multi-allelic sites**: One VCF row can contain multiple variants. You have to explode before you can merge.

None of this is insurmountable — but it's not obvious either, and it's easy to silently drop variants if you get the merge logic wrong.
